# 心情愉快地学完了第一本node教程
从 **《The Node Beginner Book》** 这本书开始入手学习Node.js, 万万没想到这本只有30多页的书， 竟然蕴含着丰富深厚的哲理， *以致于现在我实在编不下去了*。

总之， 花了一个礼拜的零散时间读完这本薄薄的也许不能称之为教程的教程（因为内容实在是太过于皮毛）， 有不少收获，对 js 和 http 的理解也更加深入, *也许我会画一个图， 描述我对 http 的理解*。

总之， 现在的我是心情愉快的。
